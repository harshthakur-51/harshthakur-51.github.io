Optical fiber ANN research

OTDR Fault Classifier

A machine-learning research project for classifying optical fiber faults from OTDR traces using a lightweight fully connected ANN.

View repositoryDiscuss this work

Role

Research model development

Context

IEEE paper under review for publication, withheld by the university's publication policy until the review process is complete

Stack

Python, ANN, OTDR traces, Edge ML, Optical networks

Reported Metrics

  • 11,706 labelled samples
  • 8 fault classes
  • up to 99.96% reported classification accuracy

Research Objective

The project explores automated fault classification for optical fiber networks so that OTDR trace interpretation does not depend entirely on manual inspection. The core problem is to detect fault patterns quickly while keeping the model lightweight enough to be practical.

Dataset and Model

A lightweight fully connected ANN was trained on 11,706 labelled samples spanning 8 fault classes. The focus of the work is not model bloat, but a compact network that can separate trace patterns with high accuracy and low overhead.

Performance Analysis

The reported experimental setup achieved 99.96% classification accuracy, which is strong for the dataset and architecture used. The project is framed as a reproducible research result, not as a claim that the model is magically perfect in every possible field condition.

Publication Status

The paper is under review, so the portfolio presents the experimental setup, the labelled class structure, and the reported accuracy without overstating deployment readiness. That keeps the work credible, which should be the bare minimum in technical writing but somehow is not.
